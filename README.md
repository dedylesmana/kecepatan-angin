# Analisis Kecepatan Angin Indonesia (2015)

Repositori ini berisi analisis statistik deskriptif dan visualisasi data
kecepatan angin di Indonesia tahun 2015 menggunakan bahasa pemrograman R.

## Isi Repositori
- Script analisis (.R)
- Dokumen R Markdown (.Rmd)
- Dataset iklim (.csv)
- Visualisasi (boxplot)

## Metode
Analisis statistik deskriptif dan visualisasi boxplot untuk melihat
distribusi kecepatan angin secara nasional dan per pulau.
